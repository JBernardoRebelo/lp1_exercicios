##Pergunta
5 - Indica o que é mostrado no ecrã pelo seguinte código C#. Justifica a tua resposta.

```cs
int n = -1;
Console.WriteLine(~n);
```

##Soluções

###Solução 1
Sabemos que: ~ | 0 = 1
				  | 1 = 0
 n = -1, ao imprimir-se o n com o método "Console.WriteLine()" utilizando-se o operador 
NOT (~) precedente à variável o que se procede em binário é o seguinte, os "1" trocam para "0":
1111 - em complemento para dois (-1)
0000 - aplicou-se o ~ e o resultado é 0, imprimindo este número no ecrã.
