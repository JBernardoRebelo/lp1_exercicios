##Pergunta
7 - Indica o que é mostrado no ecrã pelo seguinte código C#. Justifica a tua resposta.

```cs
short s = 5 / 2;
Console.WriteLine($"*\t{s}\t*");
```

##Soluções

###Solução 1
O programa vai imprimir "*	2	*"
Definem-se a a variável "s" que é a divisão de 5 por 2,
o tipo da variável é short, isso faz com que o resultado
desta divisão seja 2, a casa decimal é cortada.

Na parte em que se imprime, chama-se a variável dentro das "{}",
pois utiliza-se um "$" posterior, fazendo este chamamento possível.
Os tabs entre o dois e os asteríscos são causados pelo "\t" dentro
da string.
