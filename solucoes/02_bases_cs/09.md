##Pergunta
9 - Indica o que é mostrado no ecrã pelo seguinte código C#. Justifica a tua resposta.

```cs
char c = '\u00AE';
Console.WriteLine($@"\n{c}\n");
```

##Soluções

###Solução 1
 O programa vai imprimir "\n 00AE \n" (com o ícone correspondente
à variável). Isto acontece pois é utilizado o "@" convertendo a string
para uma string verbatim o que vai mostrar tudo dentro dela (ex: "\n")
depois sabemos que mostra o valor da variável pois é utilizado o "$"
antes da string.
