##Pergunta
8 - Indica o que é mostrado no ecrã pelo seguinte código C#. Justifica a tua resposta.

```cs
double x = 3.1223;
Console.WriteLine(@"\\{x}\\:" + x);
```

##Soluções

###Solução 1
O programa vai imprimir "\\{x}\\: 3.1223"
Esta string é verbatim, assinalada com o "@" posterior
mostrando todos os conteúdos dentro de dita string.
De seguida, executa-se uma concatenação de strings com o 
"+ x" depois de se escrever a string, adicionando o valor de 
x junto com a  string quando esta é imprimida.
