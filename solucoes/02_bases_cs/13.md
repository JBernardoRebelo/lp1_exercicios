##Pergunta
13 - Escreve um programa que solicite ao utilizador dois números reais de 64 bits
e apresente o resultado do primeiro (base) elevado ao segundo (expoente).
Sugestão: o método `Math.Pow(x, y)` realiza a operação de potenciação `xy`.
##Soluções

###Solução 1
using System;
namespace Exercicio13
{	
    class Program
    {
        static void Main(string[] args)
        {
            //declare variables
            string bas;
            string power;
            double b;
            double p;
            double result;

            //asks user for input, base number
            Console.Write("Type a number: ");
            bas = Console.ReadLine();
            b = Convert.ToInt64(bas);

            //asks user for input, power
            Console.Write("Type it's power: ");
            power = Console.ReadLine();
            p = Convert.ToInt64(power);

            //does the math we need
            result = Math.Pow(b, p);

            //outputs result to user
            Console.WriteLine(result);
        }
    }
}
